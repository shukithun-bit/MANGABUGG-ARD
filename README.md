# MangabuffCard Telegram Bot

Коллекционный Telegram gacha-бот на Python/aiogram 3 для красивых и дорогих карт Мангабафа.

## Что есть сейчас

- Общий пул карт без вселенных.
- При первом входе игрок обязан придумать ник; до этого меню не показывается.
- Один инвентарь игрока с фильтрацией по рангам.
- 10 рангов карт: `E`, `D`, `C`, `B`, `G`, `P`, `A`, `S`, `X`, `Ивент`.
- Получение карты через кнопку "Получить карту".
- Если карта новая, бот пишет "Новая карта".
- Если карта уже есть, бот пишет "Вы получили повторку" и начисляет осколки.
- Очки опыта в игровой логике больше не используются.
- Ежедневный бонус крутками сохранён.
- Бесплатные крутки копятся каждый час, максимум 10 в день.
- Топ игроков считается по уникальным полученным картам.
- Отдельный топ по рефералам сохранён.
- Добавлен топ по принятым созданным картам.
- Любой пользователь может предложить карту.
- Пользовательские карты попадают в админский "Поток" на проверку.
- Админ может принять или отклонить карту из потока.
- За каждую принятую карту автор получает +1 крутку.
- При отклонении карты админ указывает причину, и она отправляется автору.
- Профиль показывает осколки, крутки, приглашения и количество полученных карт.
- Премиум и магазин пока скрыты.

## Переменные окружения

```env
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_IDS=123456789
TZ=Europe/Moscow
DB_PATH=/data/game.db
DROP_COOLDOWN_MINUTES=60
DAILY_FREE_DROP_LIMIT=10
CARD_IMAGE_DIR=/data/images/cards
REFERRAL_REWARD_SPINS=5
```

Токен бота добавляется в конце настройки проекта в локальный `.env`.

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

Для Railway используется `Procfile`:

```text
worker: python bot.py
```
